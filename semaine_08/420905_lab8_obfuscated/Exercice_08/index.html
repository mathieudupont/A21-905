Conception lumières

Ci-dessous, il y a 13 ampoules qu'on peut allumer/éteindre en cliquant dessus.

Dans chacun des 2 groupes d'ampoules, on peut aussi cliquer sur 3 boutons :

  1. Le bouton SWITCH bascule les états de toutes les ampoules.
  2. Le bouton ON allume toutes les ampoules.
  3. Le bouton OFF éteint toutes les ampoules.
