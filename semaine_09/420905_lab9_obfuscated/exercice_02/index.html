Exercice 2 - Stocker des données dans des images


Pour cet exercice, vous devrez ajoutez des attributs data- aux images pour stocker des données.

  • Appuyer sur les boutons rouges permettra d'ajouter des attributs data- dans le code HTML des images et rendre leur fond doré.
  • Le bouton vert permet d'afficher les attributs data- des images pour vérifier qu'elles ont bien été modifiées.
